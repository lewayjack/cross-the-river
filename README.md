
##Run
***
first step open the **index.html** with a browser(google chrome or firefox are all recommended).
this game will **run automatically** after you opened this page and will pick a hero at random to start.
##Play the arcade game
***
###collect gems
first you need to move  hero on the gem until ** collected enough gems**.
when it's  display **go!!!** on the top of the canvas use the arraw key to elude bugs and move to the _river_.
if you were successfully arrived the river it will display  **MISSION X!!!**,and then start the game with a random hero.
if you were hitted by the bugs it will display  **OOPS!!!**,and restart the game with a random hero.
you have **3 hero** (life) to play.after 3 heros you need to **refresh to restart**.
